# Test-QA-Automation-Engineer
Решение 
Тестовое задание для QA Automation Engineer

Вводные данные:
Главная страница сайта crtweb.ru доступная через web-браузер Mozilla firefox.
Стек технологий указан в вакансии на которую Вы откликнулись.

Задача:
Написать автоматизированный тест выполняющий следующую последовательность действий:

Открывает один из "известных" поисковиков в браузере.
Находит в поисковике сайт crtweb.ru и совершает переход на него;
Из главной страницы crtweb.ru осуществляет переход на страницу crtweb.ru/contacts;
Проверяет, что на странице присутствуют и отображены следующие данные:

Головной офис
Москва, Пресненская набережная, 12
Офис разработки
Тюмень, ул. Малыгина 84 к1, 7 этаж
+7 (499) 113-68-89
mail@crtweb.ru

При решении задачи необходимо:

Использовать только актуальные методики и практики из сферы QA и тестирование.
Придерживаться общепринятых паттернов проектирования автоматизированных тестов.
Соблюдать человеческий codeStyle.

Будет большим плюсом, если решенное задание будет выгружено в общедоступный репозиторий.

Удачи!
